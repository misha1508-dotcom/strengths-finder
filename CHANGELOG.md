# Changelog

Все значимые изменения проекта документируются в этом файле.

## [1.0.0] - 2026-01-17

### Релиз версии 1.0

Стабильная версия приложения для анализа сильных и слабых сторон.

#### Функционал

- **Анализ ситуаций**: Пользователь описывает 8-15 провальных ситуаций из жизни
- **Определение качеств**: ИИ находит проявленные качества в каждой ситуации (эмоциональные, поведенческие, когнитивные, волевые)
- **Позитивные дуалы**: Каждое негативное качество переосмысляется как сильная сторона
- **Рейтинг качеств**: Подсчёт частоты проявления каждого качества
- **Пёрышки-противовесы**: Персональные микро-действия в 3 категориях:
  - В МОМЕНТ — что делать когда "накрывает"
  - MINDSET — какую мысль держать в голове
  - РЕГУЛЯРНО — что делать раз в неделю/месяц
- **Уникальные действия**: 3 неочевидных и элегантных совета именно для этого человека
- **Роли где будет легко**: 10 профессий/ролей, где человеку будет комфортно
- **Как капитализировать качества**: 5 советов как использовать свои особенности
- **Знаменитости**: Топ-5 известных людей с похожим типом личности (имена по-русски)
- **Хобби для души**: 5 занятий для самореализации
- **Экспорт данных**: Копирование всей информации для использования в других нейросетях

#### Технические изменения

- Объединённый API-запрос `feathersAndActivities` для экономии токенов (~30-40%)
- Rate limiting: максимум 10 запросов в минуту с одного IP
- Оптимизация UI: memoization, useCallback для предотвращения лишних ререндеров
- Анимации: stagger fade-in, slide-up, scale-in, hover-lift эффекты
- Gradient text для заголовков

#### Деплой

- Домен: https://krechet.space
- Хостинг: Vercel
- Публичный доступ без авторизации (отключена SSO Protection)
- SSL-сертификат автоматически от Vercel

## Предыдущие изменения (до v1.0)

### 2026-01-17 (ранее в этот день)

#### Исправления по обратной связи

**Пёрышки**:

- Возвращён формат с 3 категориями (moment/mindset/regular) + уникальные действия
- Улучшен промпт для генерации более неочевидных и элегантных советов

**Знаменитости**:
- Убраны фотографии знаменитостей
- Теперь показывается простой список топ-5 с именами по-русски
- Формат: "Имя — почему похож"

**Хобби**:
- Восстановлена секция "Бонус: развлечения (хобби)"

#### Настройка домена

- Куплен домен krechet.space на reg.ru
- Настроены DNS записи (A-запись на 76.76.21.21)
- Привязан домен к Vercel проекту

#### Публичный доступ

- Отключена SSO Protection через Vercel API
- Теперь сайт доступен всем без авторизации

### Более ранние изменения

#### Основная архитектура

- Next.js 16 с App Router
- TypeScript для типизации
- Tailwind CSS для стилей
- OpenRouter API (Claude 3.5 Sonnet) для анализа

#### UI/UX

- Адаптивный дизайн (мобильные + десктоп)
- Тёмная и светлая тема (автоматически по системным настройкам)
- Экран загрузки с прогрессом и шуткой про ФСБ
- Таблица результатов с раскрывающимися деталями
- Кнопка доната через YooMoney
- Ссылка на Telegram для обратной связи

#### Промпты

- Список качеств по 4 категориям (эмоциональные, поведенческие, когнитивные, волевые)
- Антипаттерны для избежания банальных советов
- Примеры хороших элегантных решений в промпте
- Строгий JSON-формат ответа
