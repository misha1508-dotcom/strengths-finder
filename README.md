# Инверсия

Преврати провалы в сильные стороны. Бесплатный AI-инструмент для самопознания.

## Что это?

**Инверсия** — это психологический инструмент, который анализирует твои жизненные неудачи и находит в них скрытые качества и сильные стороны. Основан на принципе Парето: 4% усилий дают 64% результата в понимании себя.

## Как работает

1. Опиши 2-15 ситуаций, где чувствуешь провал
2. AI анализирует каждую ситуацию
3. Получи карту своих качеств и рекомендации

## Технологии

- **Frontend:** Next.js 16, React 19, TypeScript, Tailwind CSS 4
- **AI:** Claude 3.5 Sonnet (через OpenRouter)
- **Хостинг:** Vercel

## Запуск локально

```bash
npm install
npm run dev
```

Открой [http://localhost:3000](http://localhost:3000)

## Переменные окружения

Создай файл `.env.local`:

```
OPENROUTER_API_KEY=your_api_key_here
```
